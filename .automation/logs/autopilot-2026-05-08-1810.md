## Actions this fire

1. **P5 pin** (`1c0b7a0`): `testDefaultDraftStateFieldDefaultsAreFrozen`
   in `DraftEngineTests.swift` — pin every field-level default on
   `DraftEngine.DraftState()` together. The existing
   `testDefaultDraftStateIsHighConfidence` only pins the `confidence ==
   1.0 → !isLowConfidence` path; the other four fields (`text == ""`,
   `isStreaming == false`, `isDone == false`, `error == nil`) each

   composer placeholder text, spurious streaming spinner, "Send" enabled
   on a blank draft, error banner on uninitialized state. Pin all five
   together so a refactor to DraftState's defaults can't silently flip
   any.
